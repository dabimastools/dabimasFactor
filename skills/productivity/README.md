# Productivity

コード固有ではない、一般的なワークフローのツール。

## ユーザー起動（User-invoked）

自分でタイプしたときだけ起動する（`disable-model-invocation: true`）。

- **[grill-me](./grill-me/SKILL.md)** — 決定ツリーのすべての枝が解決するまで、計画や設計について徹底的にインタビューしてもらう。
- **[handoff](./handoff/SKILL.md)** — 現在の会話を引き継ぎドキュメントに圧縮し、別のエージェントが作業を継続できるようにする。

## モデル起動（Model-invoked）

モデルが自律的に呼べる（自分でタイプしても起動できる）。

- **[grilling](./grilling/SKILL.md)** — 決定ツリーのすべての枝が解決するまで、計画や設計についてユーザーを徹底的にインタビューする。
